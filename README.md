# Dictation Master - 英語ディクテーション学習アプリ

「聞いて、打って、覚える。」
ブラウザだけで手軽に本格的なディクテーション練習ができる、ゲーム感覚の学習ウェブアプリケーションです。

## 🚀 サービス概要

Dictation Masterは、リスニング力とスペリング能力を同時に向上させることを目的とした学習ツールです。
単なる暗記ではなく、ネイティブの音声を聞きながらリアルタイムでタイピングを行うことで、「音と文字の一致」を脳に定着させます。

## ターゲットユーザー
- 英語初級〜中級の学習者
- TOEICなどの試験対策として、正確なリスニング力を身につけたい方
- 隙間時間にタイピング練習を兼ねて英語を学びたい方

## ✨ 主な機能

### 1. 3つの習熟度別レベル

- 中学生レベル：日常でよく使われる基礎的な5ワード程度の英文

- 高校生レベル：関係代名詞や仮定法など、構文が複雑な英文

- TOEIC対策：ビジネスシーンやオフィス、旅行に関連する実用的な英文

### 2. インタラクティブな学習体験

- リアルタイム判定：1文字ごとに正誤を判定。ミス時には即座にサウンドで通知し、正しい入力が行われるまで次に進ませないことで、正確な綴りを意識させます。

- 1文字ヒント機能：詰まった場所を1文字ずつ開示可能。学習の挫折を防ぎ、スムーズな進行をサポートします。

- 一瞬の視覚演出：問題クリア時には「大きな赤い丸」を表示し、達成感を演出します。

- ショートカット再生：Spaceキーで英文の音声を再生できます。

### 3. 学習の振り返り（復習）機能

- 全5問終了後、今回解いた問題を一覧で表示。
- 各問題ごとの「ミスタイプ回数」と「ヒント使用回数」を可視化。
- 振り返り画面から直接ネイティブ音声を再再生し、聞き取れなかった箇所を重点的に復習できます。

## 🛠 技術的工夫と選定理由
- 本プロジェクトは、「アクセシビリティの高さ」と「パフォーマンス」にこだわり、外部サーバーやライブラリに依存しすぎない構成をとっています。
- シングルファイル構成：HTML/CSS/JavaScriptを1つのファイルに統合。ポータビリティが高く、オフライン環境やサーバーレス環境でも即座に動作します。
- Web Speech API の活用：ブラウザ標準の音声合成エンジンを使用し、幅広い英文に対応しています。
- Web Audio API によるサウンド生成：ミス音や診断BGMは周波数制御で生成し、軽量に保っています。
- 正解SEは外部音源：正解時は `ok.mp3`（プロジェクト直下に配置）を再生し、より聞き心地の良いフィードバックにしています。

- UI/UXデザイン:

    - Tailwind CSS：モダンで清潔感のあるUIを採用。

    - 単語改行の防止：学習効率を下げる「単語の途中での改行」をCSSの inline-flex と white-space：nowrap で制御。

    - ミスカウントの最適化：表示されている1文字目を誤って打ってしまった場合などはミスにカウントしないようロジックを調整し、ユーザーの操作ミスによるストレスを軽減しました。

## 🏗 技術スタック

- Logic：JavaScript (Vanilla JS / ES6+)

- APIs：Web Speech API, Web Audio API
